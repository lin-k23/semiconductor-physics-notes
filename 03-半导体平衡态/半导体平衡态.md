# 半导体平衡态

标签：#半导体平衡态 #Chapter4 #ThermalEquilibrium

来源：Chapter 4 *The Semiconductor in Equilibrium*

## 本章一句话

热平衡（thermal equilibrium）下没有外加电压（external voltage）、电场（electric field）、磁场（magnetic field）或温度梯度（temperature gradient），半导体的宏观性质不随时间变化；本章用态密度（density of states）与费米-狄拉克分布（Fermi-Dirac distribution）计算 $n_0$、$p_0$，并用掺杂（doping）与电中性条件（charge neutrality）确定费米能级（Fermi level）。

## 本章知识链

```text
g_c(E), g_v(E), f_F(E)
  -> 能量分布 n(E), p(E)
  -> 平衡浓度 n0, p0
  -> 本征半导体与 ni（intrinsic semiconductor and ni）
  -> 施主 / 受主杂质能级（donor / acceptor impurity levels）
  -> 非本征半导体（extrinsic semiconductor）
  -> 电中性条件（charge neutrality）
  -> 费米能级位置（Fermi level position）
```

## 热平衡的含义

在热平衡（thermal equilibrium）中：无外加电压、无外加电场、无外加磁场、无温度梯度，所有性质不随时间变化（time independent）。整个系统的费米能级（Fermi energy）在平衡时保持恒定（constant）。

## 本章核心问题

- 如何由 $g_c(E)f_F(E)$ 得到导带（conduction band）中的电子浓度（electron concentration）？
- 如何由 $g_v(E)[1-f_F(E)]$ 得到价带（valence band）中的空穴浓度（hole concentration）？
- 本征载流子浓度（intrinsic carrier concentration）$n_i$ 为什么对温度极其敏感？
- 施主 / 受主（donor / acceptor）如何改变多数载流子（majority carrier）与费米能级？
- 电中性条件（charge neutrality）如何决定掺杂半导体（doped semiconductor）的平衡载流子浓度？
