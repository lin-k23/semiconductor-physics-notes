# 半导体物理复习笔记

这个仓库用于存储「半导体物理」课程的复习笔记，采用 **Obsidian Markdown** 格式组织，方便之后统一迁入本地 Obsidian Vault。

## 使用方式

- 每个知识点尽量拆成独立 Markdown 文件，方便双向链接。
- 重要概念使用 `[[概念名]]` 建立内部链接。
- 公式、推导、易错点、例题尽量分区记录。
- 总览页从 [[00-总览/半导体物理复习路线]] 开始。

## 目录结构

```text
00-总览/          复习路线、知识地图、术语索引
01-晶体结构与能带/  晶格、倒格子、能带、有效质量
02-载流子统计/      态密度、费米分布、载流子浓度
03-输运现象/        漂移、扩散、迁移率、连续性方程
04-PN结/            平衡 PN 结、偏置、耗尽区、电容
05-MOS结构/         MOS 电容、阈值电压、表面势
06-半导体器件/      二极管、BJT、MOSFET、光电器件
99-模板/            概念卡片、公式整理、例题复盘模板
```
