# PN结

标签：#PN结 #pnJunction #Chapter7

来源：Chapter 7 *The pn Junction*

## 本章一句话

PN 结（pn junction）是 p 型半导体（p-type semiconductor）和 n 型半导体（n-type semiconductor）在同一单晶材料中相邻形成的结；本章研究它在零偏（zero bias）和反向偏置（reverse bias）下的空间电荷区、内建电势、电场、结电容和击穿行为。

## 本章知识链

```text
p 区 / n 区接触
  -> 冶金结（metallurgical junction）
  -> 多数载流子扩散（majority carrier diffusion）
  -> 固定离化施主 / 受主电荷
  -> 空间电荷区 / 耗尽区（space charge / depletion region）
  -> 内建电场（built-in electric field）
  -> 内建电势 Vbi（built-in potential）
  -> 零偏耗尽宽度 W
  -> 反向偏置下 W、Emax、Cj 的变化
  -> 雪崩击穿 / 齐纳击穿
  -> 非均匀掺杂结与变容二极管
```

## 本章核心图像

1. p 区和 n 区接触后，多数电子从 n 区向 p 区扩散，多数空穴从 p 区向 n 区扩散。
2. 扩散离开后的离化施主（ionized donors）和离化受主（ionized acceptors）留下固定电荷。
3. 固定电荷形成空间电荷区 / 耗尽区（space charge region / depletion region）。
4. 空间电荷区中的电场方向从 n 区指向 p 区。
5. 热平衡下扩散趋势和电场力平衡，因此总电流为零。
