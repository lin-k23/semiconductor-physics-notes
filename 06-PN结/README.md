# 06-PN结

标签：#章节入口 #PN结 #Chapter7

## 本章目标

理解 PN 结（pn junction）的静电学：从 p 型区（p-type region）和 n 型区（n-type region）接触形成冶金结（metallurgical junction），到空间电荷区 / 耗尽区（space charge region / depletion region）、内建电势（built-in potential）、电场分布（electric field）、反向偏置（reverse bias）、结电容（junction capacitance）、击穿（breakdown）和非均匀掺杂结（nonuniformly doped junction）。
