# 光电器件总览

标签：#光电器件 #Optoelectronics #太阳能电池 #光探测器 #LED #激光二极管 #Chapter14

来源：Chapter 14 *Optical Devices*

## 本章一句话

光电器件（optoelectronic devices）利用半导体中光子与电子的相互作用，实现光能与电能之间的转换：太阳能电池和光探测器把光信号变成电信号，发光二极管和激光二极管把电注入变成光输出。

## 本章简化知识链

```text
光子能量 hν 与带隙 Eg 比较
  -> 光吸收产生电子-空穴对
  -> 太阳能电池 / 光探测器：内建电场或外加电场收集光生载流子
  -> LED：正向注入后自发辐射复合
  -> 激光二极管：粒子数反转 + 光学谐振腔 + 受激发射
```

## 本章不作为重点时的复习抓手

- 记住光吸收条件：$h\nu\gtrsim E_g$ 才能有效产生电子-空穴对。
- 记住太阳能电池等效为“光生电流源 + 二极管”。
- 记住光电二极管一般反偏工作，光电流近似正比于入射光强。
- 记住 LED 靠自发辐射，激光二极管靠受激辐射和光学反馈。
- 记住直接带隙材料更适合高效率发光。
