# 双极型晶体管总览

标签：#BJT #双极型晶体管 #晶体管 #Chapter12

来源：Chapter 12 *The Bipolar Transistor*

## 本章一句话

双极型晶体管（bipolar junction transistor, BJT）是由两个相互作用的 PN 结构成的三端有源器件；在正向有源区（forward-active mode）中，基极-发射极结正偏负责注入少数载流子，基极-集电极结反偏负责收集少数载流子，从而实现“用 $v_{BE}$ 控制 $i_C$”。

## 本章知识链

```text
npn / pnp 三层结构
  -> 发射极重掺杂、基区很窄、集电区较轻掺杂
  -> B-E 正偏注入少数载流子
  -> B-C 反偏扫出少数载流子
  -> 基区扩散电流决定集电极电流
  -> 共基极电流增益 alpha 与共发射极电流增益 beta
  -> 非理想效应：基区复合、发射效率、Early 效应、高注入、击穿
  -> 小信号模型、频率响应、开关瞬态
```

## 本章核心问题

- 为什么 BJT 需要“重掺杂发射极 + 窄基区 + 较轻掺杂集电极”？
- 为什么 $i_C$ 主要由 $v_{BE}$ 控制，而对 $v_{CB}$ 近似不敏感？
- 为什么基区宽度必须小于少数载流子扩散长度？
- 共基极电流增益（common-base current gain）$\alpha$ 和共发射极电流增益（common-emitter current gain）$\beta$ 的物理含义是什么？
- 截止、正向有源、饱和、反向有源四种工作模式如何由两个结的偏置判断？
- Early 效应、基区复合、高注入和击穿如何破坏理想模型？
- 小信号模型中 $g_m$、$r_\pi$、$r_o$ 和结电容分别来自什么物理过程？
