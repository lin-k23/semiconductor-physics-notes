# pnJFET与MESFET工作图像

标签：#pnJFET #MESFET #耗尽区 #夹断 #Chapter13

## 一句话理解

pn JFET 和 MESFET 都是在半导体中预先存在一个导电沟道，然后用反向栅压扩大耗尽区来“挤窄”沟道；沟道越窄，电阻越大，漏电流越小。

## pn JFET 基本结构

以 n 沟道 pn JFET 为例：

```text
p+ 栅区 / n 型沟道 / p+ 栅区
源极：电子进入沟道
漏极：电子流出沟道
栅极：反偏 PN 结，控制耗尽区宽度
```

当 $V_{GS}=0$ 且 $V_{DS}$ 很小时，沟道像普通 n 型电阻，$I_D$ 与 $V_{DS}$ 近似线性。

## 栅压控制沟道

对 n 沟道 pn JFET，施加负栅压：

```text
V_GS 更负
  -> 栅-沟道 PN 结反偏增大
  -> 耗尽层伸入 n 沟道
  -> 有效沟道厚度变小
  -> 沟道电阻增大
  -> I_D 下降
```

当耗尽层完全填满沟道时，称为夹断（pinchoff），器件近似关断。

> [!figure] Fig-13-3 占位
> 不同栅压下的栅-沟道耗尽区和小 $V_{DS}$ I-V 曲线。

## 漏压导致的漏端夹断

即使 $V_{GS}=0$，当 $V_{DS}$ 增大时，沟道电势从源到漏逐渐升高，漏端栅-沟道结反偏更强，漏端耗尽层更宽。

```text
V_DS 增大
  -> 漏端反偏更强
  -> 漏端沟道更窄
  -> 沟道电阻沿 x 方向变化
  -> 漏端先发生夹断
```

漏端夹断后，电流不会变成零。载流子先通过剩余沟道到达夹断点，再被夹断区电场扫入漏极。理想情况下继续增大 $V_{DS}$，$I_D$ 近似饱和。

## MESFET 工作图像

MESFET（metal-semiconductor field-effect transistor）把 pn JFET 的 PN 栅换成肖特基势垒栅（Schottky barrier gate）。常见结构是 GaAs MESFET：

- n 型 GaAs 外延层作为沟道。
- 半绝缘 GaAs 衬底降低寄生电容。
- 源漏为欧姆接触。
- 栅极为肖特基整流接触。

> [!figure] Fig-13-6 占位
> n 沟道 GaAs MESFET 截面图。

肖特基栅反偏时，金属下方形成耗尽区并调制沟道厚度。若沟道在 $V_{GS}=0$ 已经导通，就是耗尽型 MESFET；若 $V_{GS}=0$ 沟道已经被耗尽，需要正栅压打开沟道，就是增强型 MESFET。

> [!figure] Fig-13-8 占位
> 增强型 MESFET 在 $V_{GS}=0$、$V_{GS}=V_T$ 和 $V_{GS}>V_T$ 时的沟道耗尽状态。

## pn JFET 与 MESFET 对比

| 项目 | pn JFET | MESFET |
|---|---|---|
| 栅结构 | 反偏 PN 结 | 反偏肖特基势垒结 |
| 常见材料 | Si | GaAs 等 III-V 材料 |
| 沟道控制 | PN 结耗尽层 | 肖特基耗尽层 |
| 优点 | 低噪声、结构清楚 | 高迁移率、高频、寄生电容小 |
| 限制 | 速度受材料迁移率和结电容限制 | 栅正向电压受肖特基结限制 |

## 易错点

- 夹断点在漏端出现时，$I_D$ 不为零；只有栅压把整个沟道夹断时才近似关断。
- n 沟道 JFET 通常需要负栅压关断，是耗尽型器件。
- MESFET 的栅极不是绝缘栅，不能像 MOSFET 那样随意加大正栅压，否则肖特基栅会导通。
- JFET / MESFET 是多数载流子器件，因此没有 BJT 那样的少数载流子存储问题。
