# Chapter 1 图片说明

## 说明

- 图像采用 SVG 格式，便于在 GitHub 与 Obsidian 中直接渲染。
- 内容参考上传 PDF 的 Chapter 1：The Crystal Structure of Solids。
- 图中主要关键词保留英文，例如 `unit cell`、`Miller indices`、`diamond lattice`、`zincblende`。
- 当前版本以学习笔记用的重绘示意图为主，而不是逐像素复制教材原图；这样更适合长期维护与二次编辑。
