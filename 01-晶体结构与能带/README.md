## 与后续章节的连接

- 晶体周期性会进入周期势场（periodic potential），后续在 [[02-量子力学初步/能带形成与Kronig-Penney模型]] 中用于解释能带形成。
- 晶面（crystal plane）和晶向（crystal direction）会影响晶圆取向、器件版图与工艺窗口。
- 缺陷（defects）和杂质（impurities）会影响迁移率（mobility）、复合中心（recombination centers）和 [[05-非平衡载流子/过剩载流子寿命与SRH复合]]。
- 材料生长（crystal growth）和外延生长（epitaxial growth）决定器件层质量，是后续 PN 结、MOS 结构和光电器件的工艺基础。

- 晶体周期性如何影响电子运动？

- 如何由截距求密勒指数（Miller indices）？
- 晶面 `(hkl)` 与晶向 `[hkl]` 有什么区别？
- 为什么硅（Si）是金刚石结构（diamond structure）？
- 砷化镓（GaAs）的闪锌矿结构（zincblende lattice）与硅的金刚石结构有何异同？
- 缺陷（defects）与杂质（impurities）如何影响电学性质？
- 直拉法（Czochralski method）与外延生长（epitaxial growth）分别解决什么问题？
