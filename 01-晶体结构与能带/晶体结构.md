# 晶体结构

标签：#晶体结构 #Chapter1 #总览

来源：Chapter 1 *The Crystal Structure of Solids*

## 本章一句话

半导体的电学性质不仅由化学组成决定，也强烈依赖原子在固体中的排列方式；对器件而言，理解单晶（single crystal）、晶胞（unit cell）、密勒指数（Miller indices）、金刚石结构（diamond structure）和缺陷（defects）是后续学习能带、载流子输运与 PN 结的基础。

## 本章知识链

```text
半导体材料（semiconductor materials）
  → 固体类型（types of solids）
  → 晶格 / 晶胞（lattice / unit cell）
  → SC / BCC / FCC
  → 晶面 / 密勒指数（crystal planes / Miller indices）
  → 晶向（crystal directions）
  → 金刚石 / 闪锌矿结构（diamond / zincblende structure）
  → 原子键合（atomic bonding）
  → 缺陷 / 杂质 / 掺杂（defects / impurities / doping）
  → 晶体生长 / 外延（crystal growth / epitaxy）
```

## 和后续章节的连接

- 金刚石结构（diamond structure）决定 Si / Ge 的四面体共价键结构，是理解 [[02-量子力学初步/能带形成与Kronig-Penney模型]] 的材料背景。
- 密勒指数（Miller indices）用于描述晶圆表面取向，例如 `(100)` wafer、`(111)` plane。
- 缺陷（defects）与杂质（impurities）会引入散射（scattering）、复合中心（recombination centers）或受主 / 施主能级，是理解 [[03-半导体平衡态/掺杂与杂质能级]]、[[04-输运现象/迁移率与散射]] 和 [[05-非平衡载流子/过剩载流子寿命与SRH复合]] 的前置知识。
- 外延生长（epitaxial growth）决定器件表面层质量，与 PN 结、MOS、BJT 和光电器件工艺紧密相关。

## 复习优先级

1. 先会区分非晶（amorphous）、多晶（polycrystalline）、单晶（single crystal）。
2. 再掌握晶胞（unit cell）、原胞（primitive cell）、晶格常数（lattice constant）。
3. 必须熟练密勒指数（Miller indices）的求法。
4. 重点记住 Si 的金刚石结构（diamond structure）：每个原子有 4 个最近邻（nearest neighbors）。
5. 理解掺杂（doping）是通过受控杂质改变半导体导电性。
