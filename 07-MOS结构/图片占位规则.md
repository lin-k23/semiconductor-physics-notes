# 图片占位规则

标签：#整理规则 #图片占位 #NotesOnly

## 当前规则

后续整理优先采用 notes-only 工作流。必要示意图只在 Markdown 中保留占位，不上传截图、不裁剪图片、不插入图片链接。

占位格式：

```markdown
> [!figure] Fig-10-4 占位
> p 型衬底 MOS 电容的积累、耗尽、反型能带图。后续插入 PDF 原图截图。
```

其中 `Fig-x-y` 表示 Chapter x 的 Figure x.y，例如 `Fig-11-5` 表示 Chapter 11 Figure 11.5。

## 后续处理

截图、裁剪、图片文件上传和 `_assets/pdf-screenshots/README.md` 维护后续单独处理。笔记 PR 中只保留占位文字，避免因图片处理拖慢整理速度。
