# MOSFET击穿与穿通

标签：#MOSFET #击穿 #穿通 #LDD #Chapter11

## 一句话理解

MOSFET 的击穿和穿通来自短沟道、高电场和寄生结构：漏端高反偏可引发雪崩倍增，源漏耗尽区接近相连会导致穿通，而 LDD 结构通过降低漏端峰值电场缓解这些问题。

## 漏-衬底结击穿

n 沟道 MOSFET 中，漏区 n+ 与 p 型衬底形成反偏 PN 结。当 $V_{DS}$ 很大时，漏端空间电荷区电场增大，可能发生雪崩击穿（avalanche breakdown）。

```text
V_DS 增大
  -> 漏-衬底结反偏增大
  -> 漏端峰值电场增大
  -> 碰撞电离产生电子-空穴对
  -> 漏电流和衬底电流增大
```

## 寄生双极晶体管效应

碰撞电离产生的空穴流向衬底，会在衬底电阻上产生压降。如果该压降使源-衬底结被正向偏置，就可能触发寄生 npn 双极晶体管，进一步放大漏电流。

物理链：

```text
漏端碰撞电离
  -> 空穴流过衬底电阻
  -> 源-衬底结局部正偏
  -> 寄生 BJT 开始导通
  -> 击穿电流增强
```

## 穿通与近穿通

穿通（punch-through）指源端和漏端耗尽区相连，栅极失去对源端势垒的有效控制。短沟道中，实际近穿通常在完全耗尽区相连之前就显著出现，表现为 DIBL 和关断漏电增大。

## LDD 结构

轻掺杂漏（lightly doped drain, LDD）在沟道和重掺杂漏区之间加入一段轻掺杂 n 区，使漏端电场分布更平缓。

```text
传统漏结构
  -> 高掺杂漏端电场尖峰强

LDD 结构
  -> 电场分布拉宽
  -> 峰值电场降低
  -> 雪崩、热电子、氧化层注入风险降低
```

## 代价

LDD 会增加：

- 工艺复杂度。
- 漏端串联电阻。
- 导通电阻和功耗。

因此 LDD 是可靠性和导通性能之间的折中。

## 易错点

- MOSFET 击穿不只是 PN 结雪崩，还可能被寄生 BJT 放大。
- 穿通是源漏势垒控制问题，不等同于氧化层击穿。
- LDD 降低峰值电场，但会增加串联电阻。
- 短沟道器件即使没有完全穿通，也可能已有明显 DIBL 和亚阈值漏电。
