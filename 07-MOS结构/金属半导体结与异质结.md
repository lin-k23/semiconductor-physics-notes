# 金属半导体结与异质结

标签：#金属半导体结 #肖特基二极管 #欧姆接触 #异质结 #Chapter9

来源：Chapter 9 *Metal–Semiconductor and Semiconductor Heterojunctions*

## 本章一句话

金属-半导体结（metal–semiconductor junction）决定器件如何与外部电路接触：它可以形成整流的肖特基势垒二极管（Schottky barrier diode），也可以通过低势垒或隧穿形成欧姆接触（ohmic contact）；异质结（heterojunction）则通过不同带隙材料的能带不连续实现势阱、载流子限制和二维电子气（two-dimensional electron gas, 2-DEG）。

## 本章知识链

```text
金属功函数 + 半导体电子亲和势
  -> 接触后费米能级对齐
  -> 能带弯曲 + 空间电荷区
  -> 肖特基势垒高度与内建势
  -> 热电子发射主导的 I-V
  -> 势垒降低 / 界面态导致非理想
  -> 欧姆接触：低势垒或高掺杂隧穿
  -> 异质结：带阶、势阱、二维电子气
```

## 本章核心问题

- 为什么金属和 n 型半导体接触后会出现耗尽区和势垒？
- 肖特基势垒高度（Schottky barrier height）由哪些量决定，为什么实际值会偏离理想值？
- 肖特基二极管和 PN 结二极管的 I-V 形式相似，为什么导通电压和开关速度不同？
- 欧姆接触（ohmic contact）为什么强调低接触电阻，而不是只看是否“能导通”？
- 高掺杂为什么能把整流接触变成隧穿接触（tunneling contact）？
- 异质结（heterojunction）中的导带不连续（conduction-band discontinuity）和价带不连续（valence-band discontinuity）如何决定载流子限制？
- 二维电子气（2-DEG）为什么是“垂直方向受限、平面内自由”的电子系统？
