# CMOS技术基础

标签：#CMOS #NMOS #PMOS #反相器 #Chapter10

## 一句话理解

CMOS（complementary MOS）把 n 沟道 MOSFET 和 p 沟道 MOSFET 互补使用，使逻辑门在静态 0 / 1 状态下理想直流电流近似为零，从而实现低静态功耗。

## 互补器件思想

CMOS 的基本单元是反相器：

```text
输入低电平
  -> pMOS 导通，nMOS 截止
  -> 输出被拉高

输入高电平
  -> nMOS 导通，pMOS 截止
  -> 输出被拉低
```

因为任一稳定逻辑状态下，电源到地之间理想上没有直流导通通路，所以静态功耗很低。

## 为什么需要互补沟道

单独 NMOS 负载逻辑在某些状态下会存在直流通路。CMOS 用 pMOS 作为上拉网络，用 nMOS 作为下拉网络：

- pMOS 负责把输出拉向 $V_{DD}$。
- nMOS 负责把输出拉向地。
- 两者由同一个输入互补控制。

## 和器件物理的连接

CMOS 工艺中要同时实现：

- n 沟道 MOSFET：通常在 p 型阱或 p 型衬底中形成。
- p 沟道 MOSFET：通常在 n 型阱中形成。
- 低阻欧姆接触：源漏重掺杂区需要与金属形成低接触电阻。
- 合理阈值电压：通过栅材料、氧化层厚度、衬底掺杂和离子注入控制。

## 动态功耗

CMOS 的主要功耗常来自电容充放电：

$$
P_{dyn}\approx \alpha C_L V_{DD}^2 f
$$

其中 $\alpha$ 是开关活动因子，$C_L$ 是负载电容，$f$ 是开关频率。

## 易错点

- CMOS 低功耗指静态功耗低，不代表动态功耗为零。
- 实际 CMOS 有亚阈值漏电、栅漏电和结漏电，见 [[亚阈值导电]]。
- pMOS 迁移率低于 nMOS，实际版图常用更大的 pMOS 宽度补偿驱动能力。
- CMOS 速度不仅由 MOSFET 本征速度决定，还受互连和负载电容影响。
