# MOSFET缩放

标签：#MOSFET缩放 #常电场缩放 #CMOS #Chapter11

## 一句话理解

MOSFET 缩放（MOSFET scaling）的目标是在缩小沟道长度提高密度和速度的同时，控制电场、功耗和可靠性；理想常电场缩放要求尺寸和电压按同一比例缩小，掺杂按反比例提高。

## 常电场缩放思想

设缩放因子 $k<1$。常电场缩放（constant-field scaling）要求器件尺寸和电压同时缩小，使横向和垂直电场保持近似不变。

```text
L -> kL
W -> kW
t_ox -> kt_ox
x_j -> kx_j
V -> kV
N_A, N_D -> N_A/k, N_D/k
```

这样可以避免因尺寸缩小而电场上升导致击穿、热载流子和氧化层可靠性问题。

## 常电场缩放结果

典型关系：

| 参数 | 缩放趋势 |
|---|---|
| 器件尺寸 $L,W,t_{ox}$ | $k$ |
| 掺杂浓度 | $1/k$ |
| 电压 | $k$ |
| 电场 | 约不变 |
| 电容 | $k$ |
| 单器件电流 | $k$ |
| 单器件功耗 | $k^2$ |
| 器件面积 | $k^2$ |
| 器件密度 | $1/k^2$ |
| 延迟时间 | $k$ |
| 功耗密度 | 约不变 |

## 阈值电压为什么难缩放

理想阈值电压：

$$
V_T=V_{FB}+2\phi_f+\frac{\sqrt{4e\varepsilon_sN_A\phi_f}}{C'_{ox}}
$$

其中 $V_{FB}$ 和 $2\phi_f$ 是材料和掺杂相关项，并不会简单按 $k$ 缩放。因此阈值电压不能像电源电压那样无条件同比例缩小。

## 广义缩放

实际技术演进中，电源电压常没有完全按尺寸同比例降低，原因包括：

- 兼容已有电源标准。
- 阈值电压不能无限降低，否则亚阈值漏电太大。
- 电路噪声裕量需要一定电压摆幅。

这导致实际器件中的电场往往随缩小而升高，从而带来：

- 氧化层可靠性下降。
- 热电子效应增强。
- 速度饱和更明显。
- 功耗密度和散热问题更严重。

## 易错点

- 缩小 $L$ 会提高速度，但不是只缩 $L$ 就行；电压、氧化层、结深和掺杂都必须配合。
- 常电场缩放是理想规则，实际 CMOS 缩放常偏离它。
- 阈值电压不能随意降低，亚阈值导电是基本限制。
- 缩放提升密度，但也引入短沟道、热载流子和氧化层隧穿等可靠性问题。
